Welcome to Gemini Grammar Helper!

1 Get your Gemini API key

Visit Google AI Studio to create your API key. If you don't have an account, you'll need to create one first.

2 Enter your API key below

This key will be stored securely in your browser and is needed for the extension to work.

3 Start using the extension

Once your API key is saved, you can start using the extension:

  • Click into any text field on a webpage
  • Look for the "G" button that appears
  • Click it to see grammar and clarity improvement options
